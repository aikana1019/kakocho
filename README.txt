============================================================
  デジタル過去帳アプリ  README
============================================================

仏壇の前にスマホ・タブレットを置いて、毎日の回向に使える
ブラウザだけで動く、インストール不要の過去帳アプリです。


【特徴】
  ・インストール不要  — kakocho.html をブラウザで開くだけ
  ・オフライン動作   — ネット接続不要（フォント初回読込時のみ必要）
  ・全宗派対応      — 中央のお題目を自由に変更可能
  ・プライバシー安心 — データはすべて端末内に保存。外部送信なし
  ・マルチデバイス  — スマホ・タブレット・PC すべてで動作


【動作環境】
  PC（Windows/Mac）： Chrome / Edge / Firefox / Safari
  iPhone / iPad    ： Safari
  Android          ： Chrome


============================================================
  基本的な使い方
============================================================

1. kakocho.html をブラウザで開く
2. 表紙をタップ／クリックして開く
3. ◀ ▶ ボタン、または左右スワイプ（スマホ）でページ移動
   ※ キーボードの矢印キー、テンキーの 4（前日）6（翌日）も使えます


【記録の登録】
  下のメニューの「＋ 記録」をタップして入力します。

  御仏名（戒名）  空欄の場合は俗名を代わりに大きく表示します
  俗名            生前のお名前
  享年・続柄      例：享年七十三歳　父
  没年            例：令和7（回忌の自動計算に使用）
  備考            自由記入（複数行対応）


【回忌の自動表示】
  没年を入力しておくと、祥月命日のページに
  「十三回忌」などのバッジが自動で表示されます。

  対応回忌：
    一周忌 / 三回忌 / 七回忌 / 十三回忌 / 十七回忌 /
    二十三回忌 / 二十七回忌 / 三十三回忌 / 三十七回忌 /
    五十回忌 / 百回忌


【没年の入力形式】
  西暦・和暦どちらでも入力できます。
    例：2024  または  令和6
    対応元号：令和・平成・昭和・大正・明治


============================================================
  設定
============================================================

「⚙ 設定」メニューから変更できます。

  家名          表紙・CSVファイル名に使用（例：山田）
  お題目・宗派名 中央に表示するお題目を変更


【宗派別お題目の例】
  日蓮宗・創価学会       南無妙法蓮華経
  浄土宗・浄土真宗       南無阿弥陀仏
  真言宗                南無大師遍照金剛
  曹洞宗・臨済宗（禅宗） 南無釈迦牟尼仏
  天台宗                南無妙法蓮華経 または 南無阿弥陀仏


============================================================
  背景設定
============================================================

「🖼 背景」メニューから変更できます。

  ・カラープリセット8種（漆黒・夜桜・枯山水・深緑・深海・木目・朱・石墨）
  ・写真を選ぶ — お仏壇・庭・季節の風景などを設定
  ・設定した背景は次回起動時も保持されます


============================================================
  データのバックアップ
============================================================

「⚙ 設定」→「CSVファイルを保存」でバックアップできます。

  ・すべての記録をCSVファイルに書き出し
  ・「CSVから読み込み」で別の端末への移行も可能
  ・重複データは「重複データを削除」で一括整理


============================================================
  ページの見方
============================================================

  【上部エリア】 本日の行事・記念日

  【下部 右ページ（当日）】 ／ 【左ページ（翌日）】
    各列に 戒名または俗名（大きく）
          享年・続柄（列の下側）
          備考（列の下側）
    が縦書きで表示されます。

  【色の意味】
    赤ラベル「祥月命日」 — 命日と同じ月日のページ
    茶ラベル「月命日」   — 命日と同じ日（月が違う）のページ
    赤バッジ「十三回忌」 — 今年が回忌の年


【固定行事（初期設定）】
  2月16日  日蓮大聖人御降誕
  4月28日  立教開宗会
  10月13日 御会式
  11月18日 創価学会創立記念日

  ※「🔔 行事」メニューから追加・削除できます


============================================================
  スマホのホーム画面に追加する方法
============================================================

iPhone（Safari）：
  1. Safari で kakocho.html を開く
  2. 下部の共有ボタン（□から矢印が出るアイコン）をタップ
  3. 「ホーム画面に追加」を選択
  → アプリのように起動できます

Android（Chrome）：
  1. Chrome で kakocho.html を開く
  2. 右上のメニュー（⋮）をタップ
  3. 「ホーム画面に追加」を選択


============================================================
  プライバシーについて
============================================================

  ・すべてのデータはお使いの端末内にのみ保存されます
  ・外部サーバーへのデータ送信は一切ありません
  ・インターネットはフォント（Google Fonts）の
    初回読込時のみ使用します


============================================================
  CSVデータ形式
============================================================

  月,日,戒名,俗名,享年,没年,備考
  12,13,,上田 誠一郎,享年 ７３歳,令和7,雅子さんの父上
