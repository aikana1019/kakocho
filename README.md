# 📖 デジタル過去帳アプリ

> 仏壇の前にスマホ・タブレットを置いて、毎日の回向に使える  
> ブラウザだけで動く、インストール不要の過去帳アプリです。

---

## ✨ 特徴

- **インストール不要** — HTMLファイルをブラウザで開くだけ
- **オフライン動作** — インターネット接続不要（フォント初回読込時のみ必要）
- **全宗派対応** — 中央のお題目を自由に変更可能
- **データはブラウザに保存** — サーバー不要、個人情報の外部送信なし
- **スマホ・タブレット・PC** すべてで動作

---

## 🚀 使い方

### 基本操作

1. `kakocho.html` をブラウザで開く
2. 表紙をタップ／クリックして開く
3. **◀ ▶ ボタン** または **左右スワイプ**（スマホ）でページ移動
4. キーボードの矢印キー・テンキーの `4`（前日）`6`（翌日）でも移動可能

### 記録の登録

1. 下のメニューから **「＋ 記録」** をタップ
2. 以下の項目を入力して「保存」

| 項目 | 説明 |
|------|------|
| 御仏名（戒名） | 空欄の場合は俗名を代わりに表示 |
| 俗名 | 生前のお名前 |
| 享年・続柄 | 例：享年七十三歳　父 |
| 没年 | 例：令和7（回忌の自動計算に使用） |
| 備考 | 自由記入（複数行対応） |

### 回忌の自動表示

没年を入力しておくと、祥月命日のページに **「十三回忌」** などのバッジが自動表示されます。

対応回忌：一周忌 / 三回忌 / 七回忌 / 十三回忌 / 十七回忌 / 二十三回忌 / 二十七回忌 / 三十三回忌 / 三十七回忌 / 五十回忌 / 百回忌

### 没年の入力形式

西暦・和暦どちらでも入力できます。

```
2024       ← 西暦
令和6      ← 和暦（令和・平成・昭和・大正・明治 に対応）
```

## ⚙️ 設定

**「⚙ 設定」** メニューから変更できます。

| 設定項目 | 説明 |
|---------|------|
| 家名 | 表紙・CSVファイル名に使用（例：山田） |
| お題目・宗派名 | 中央に表示するお題目を変更 |

### 宗派別お題目の例

| 宗派 | お題目 |
|------|-------|
| 日蓮宗・創価学会 | 南無妙法蓮華経 |
| 浄土宗・浄土真宗 | 南無阿弥陀仏 |
| 真言宗 | 南無大師遍照金剛 |
| 曹洞宗・臨済宗（禅宗） | 南無釈迦牟尼仏 |
| 天台宗 | 南無妙法蓮華経 または 南無阿弥陀仏 |

---

## 🖼️ 背景設定

**「🖼 背景」** メニューから変更できます。

- **カラープリセット8種** から選択（漆黒・夜桜・枯山水・深緑・深海・木目・朱・石墨）
- **写真を選ぶ** でお仏壇・庭・季節の風景などを設定
- 設定した背景は次回起動時も保持されます（IndexedDB に保存）

---

## 💾 データのバックアップ

**「⚙ 設定」→「CSVファイルを保存」** でバックアップできます。

- すべての記録をCSVファイルに書き出し
- 「CSVから読み込み」で復元・移行可能
- 重複して読み込んだ場合は「重複データを削除」で一括整理

## 📅 表示のしくみ

### ページ構成（見開き）

```
┌──────────────────────────────────────────┐
│   上部エリア：本日の行事・記念日              │
├─────────────┬────────┬────────────────────┤
│             │  南    │                    │
│  右ページ    │  無    │   左ページ           │
│  （当日）    │  妙    │   （翌日）           │
│             │  法    │                    │
│  戒名・俗名  │  蓮    │   戒名・俗名         │
│  享年・備考  │  華    │   享年・備考         │
│             │  経    │                    │
└─────────────┴────────┴────────────────────┘
```

### 色の意味

| 色 | 意味 |
|----|------|
| 🔴 赤ラベル「祥月命日」 | 命日と同じ月日のページ |
| 🟫 茶ラベル「月命日」 | 命日と同じ日（月が違う）のページ |
| 🔴 赤バッジ「十三回忌」など | 今年が回忌の年 |

### 固定行事（初期設定）

| 日付 | 行事 |
|------|------|
| 2月16日 | 日蓮大聖人御降誕 |
| 4月28日 | 立教開宗会 |
| 10月13日 | 御会式 |
| 11月18日 | 創価学会創立記念日 |

※「🔔 行事」メニューから追加・削除・カスタマイズできます

## 📲 スマホのホーム画面に追加する方法

### iPhone（Safari）

1. Safari で `kakocho.html` を開く
2. 下部の **共有ボタン**（□から矢印が出るアイコン）をタップ
3. **「ホーム画面に追加」** を選択
4. アプリのように起動できます

### Android（Chrome）

1. Chrome で `kakocho.html` を開く
2. 右上の **メニュー（⋮）** をタップ
3. **「ホーム画面に追加」** を選択

---

## 🔒 プライバシーについて

- すべてのデータはお使いの端末内にのみ保存されます
- 外部サーバーへのデータ送信は一切ありません
- インターネットはフォント（Google Fonts）の初回読込時のみ使用します
